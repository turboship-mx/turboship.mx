# Agent Guide

This repo uses Bun for all Node tooling.

## Requirements

- Bun installed (`bun --version`)

## Common commands

```bash
bun install
bun run dev
bun run build
bun run lint
bun run preview
```

## Notes

- Use Bun to add or remove dependencies (no npm, pnpm, or yarn).
- The lockfile for this repo is `bun.lockb`.
